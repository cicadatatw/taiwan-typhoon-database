# taiwan-typhoon-database
Taiwan Typhoon Database since 1958 from CWB

## Usage

To get all typhoon details since 1958, run
```
ruby get_data.rb
```
or 
```
ruby get_data.rb all
```

To specify a year, say 2016, run
```
ruby get_data.rb 2016
```
